\# Network Scanning Labs



\## Overview



This repository contains hands-on cybersecurity labs focused on network discovery, enumeration, and traffic analysis. Each lab documents the objective, tools used, commands executed, results, and key lessons learned.



\## Skills Demonstrated



\- Network reconnaissance

\- Host discovery

\- TCP and UDP port scanning

\- Service enumeration

\- Banner grabbing

\- Operating system detection

\- Packet analysis using Wireshark

\- Technical documentation

\- Linux command-line usage



\## Tools Used



\- Nmap

\- Wireshark

\- Kali Linux

\- Windows

\- Git

\- GitHub
