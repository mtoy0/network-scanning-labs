\# Nmap Host Discovery



\## Objective



Identify live hosts on a network using Nmap host discovery techniques.



\## Environment



\- Kali Linux

\- Nmap

\- VirtualBox

\- Authorized lab network



\## Commands



```bash

nmap -sn <target-network>

```



\## Expected Results



\- Active hosts discovered

\- IP addresses identified

\- Response times observed



\## Screenshots



Screenshots of the scan results will be added after completing the lab.



\## Lessons Learned



This lab demonstrates how host discovery is performed before enumeration and vulnerability assessment. Identifying active systems is the first step in understanding the attack surface of a network.
